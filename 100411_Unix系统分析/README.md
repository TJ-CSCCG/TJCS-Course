# Unix系统分析
每周2学时，以补充《操作系统》课程为主，会深入讲解一些Linux/Unix的实现原理

**主要内容涉及：**
  * Linux内存地址
  * Linux信号处理
  * 进程
  * 内存寻址
  * 中断
  * 调度系统架构
  * 内核空间
  * 信号
  * VFS等
