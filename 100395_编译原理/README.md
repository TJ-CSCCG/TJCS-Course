# 编译原理

## 100395 编译原理

### 一、总述

#### 1. 教材

程序设计语言编译原理（第 3 版） 陈火旺 国防工业出版社

#### 2. 作业

作业大致可分为两部分，章节作业和实验。

章节作业出自课本的章节习题。实验作业分别是针对 `类 C 语言` 的 **1.1 词法分析器**、**1.2 语法分析器** 和 **2 中间代码生成器**。

#### 3. 考试

考试的难度 **小于等于** 章节作业难度，尤其是考试不会出现章节作业中出现的 **无聊、繁复** 的解题过程。对着每个知识点做一道习题，可以高效率复习！
