## 100634 随机过程

### 一、总述

#### 1. 老师

本课程可以说是同济大学的奇葩，在24级每个班级都有两位老师，作者在的是汪子乔/熊明亮老师班，另外一个班情况不清楚

汪老师，熊老师均为年轻老师，顶替了一部分杨/刘的位置吧！

#### 2. 教材

本课程没有官方教材（起码24级汪老师/熊老师班没有）

熊老师使用中科院研究生院 2009～2010 第一学期 随机过程讲稿 孙应飞，网上好像可以搜到

汪老师使用自编讲义，latex匠心手作，很用心

#### 3. 作业

汪老师原话：作业他说了不算，我们的作业应该是跟着另外一个班走

总共布置了两个小作业：

* 请选取一个生活中真实存在的随机过程现象，详细描述该随机过程现象在时间和空间上的变化特征以及具体表现，深入分析该随机性的核心来源（不少于 三 点），详细说明该随机现象是否存在可观察的统计规律。要求内容具体，逻辑清晰，回答不少于1200字。

* 选取一个你感兴趣的经典随机过程模型，展开全面且深入的分析，字数不少于 1500 字。具体要求如下。
1、清晰且系统地阐述所选模型的核心定义，数学表达形式，关键假设条件及核心特征，不要仅仅简单罗列概念。
2、选择一个具体的生活或行业应用场景，分析该数学模型与场景的适配性。详细描述选取场景的具体背景，核心问题，以及场景中随机过程的表现形式。说明为何所选模型适合拟合该场景的随机过程。
3、理论结合实际，全面分析模型在该场景中应用的合理性与局限性。

一个大作业：

见文件

#### 4. 课堂

汪老师名言：如果你不能记住这辈子学习到的所有数学知识，那你别来学这门课，还好你们只是可爱的本科生，别担心太多。

汪老师讲课很生动，明显能感觉出来汪老师对讲授的内容了如指掌，而且给他发邮件也会回你，第一第二节课用三门问题等进行引入。

汪老师习惯在课后进行小测代替签到，如果你没听，那肯定是不会的，ai基本也是乱答，当然因为课程难度较大，听了也未必会。

汪老师讲义应该都是自己编写的，全英文版本的，很有内容。

个人认为，这是大学阶段学到现在最难也是最有意思的数学课之一，推荐听一听。

熊老师：熊老师讲课水平一般，基本上是照着讲义念，其实汪老师第一第二节课还是不少人听的，但是之后几节课都是熊老师上，最后几节才换成汪老师，导致后面也没啥人听了。

课堂上基本只有个位数的人在听讲。

#### 5. 考试

> 没有。全靠大作业，如果你想得优，那么去卷报告吧！听说大把卷到七十多页的，笔者写了十几页的报告吃良了。

不过，汪老师说如果他掌权，肯定改成考试课。
