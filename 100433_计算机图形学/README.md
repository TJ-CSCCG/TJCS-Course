# 计算机图形学

## 1000433 计算机图形学

### 一、总述

#### 1. 教材
参考教材使用：Computer Graphics with Open GL. by Donald D. Hearn

仅作参考书（字典）使用，书中有详细的图形学算法的opengl实现。

#### 2. 作业

作业组成为：4次小作业+大作业

小作业：

+  每次小作业由若干个概念阐释题目和一个编程题目构成，编程题不要求提交源代码，只需要展示关键代码和成品结果。
+  要求用OpenGL实现，考察最基本的图形学算法，在熟悉api的前提下不会占用很多时间。

大作业：

+   自由选择主题，如果偏向渲染可以做PBR的各种材质、进阶光照算法、体积渲染等，如果偏向仿真可以做流体的模拟，同时也有较多小组选择做游戏，综合性更强。
+   大作业的“官方”启动日期为第九周（防止组员中期退课）。分为：开题展示(proposal)，中期展示，和期末展示三个阶段。建议提前开始规划，提前开始组队，临近期末时间会非常紧张。
+   如果想在大作业中加入物理引擎，建议找成熟的引擎方案，即文档多，bug少的。血与泪的故事orz 
+   其他：建议参考CS课程指北.md中关于图形学的介绍，本文档和课程指北中的描述是由两位同学创建的。
